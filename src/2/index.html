产品优势

  • 可用性

    通过冗余设计，自动移除异常节点，并将流量在正常节点之间重新路由，可用性高达99.95%。

  • 性能

    支持最高10万并发连接，满足用户的大流量需求；支持四层负载均衡（包含 TCP 协议和 UDP 协议）和七层负载均衡（包含 HTTP 协议和 HTTPS协议）。

  • 经济

    用户无需再购买负载均衡硬件，节省了硬件的投入及维护成本。

  • 弹性

    根据应用的流量，自动扩展处理能力，并可与弹性伸缩服务无缝集成，自动满足变化的流量需求。

    • 产品功能
  • 多协议转发
    支持用户使用4层（TCP协议）或7层（HTTP协议、HTTPS协议）的负载分发； HTTPS协议提供集中化证书管理服务；
  • 自动扩展
    根据用户的实际流量，自动扩展负载分发能力。
  • 健康检查
    通过多种方法对后端云主机的运行状态进行检查，自动移除异常实例，恢复正常后，再继续进行转发，确保业务可用性。
  • 会话保持
    对TCP和HTTP协议都支持会话保持功能，将同一个用户的请求分发到后端同一个服务器上。
  • 实时监控
    提供实例、流量、连接数和数据包等实时监控信息，方便用户随时掌握服务状况。
    • 应用场景
  • 构建高可用服务
    云主机配合负载均衡一起使用，通过部署多主机提高服务的可用性，无需程序修改，即可方便的实现高可用方案。
  • 业务弹性伸缩
    业务统一通过负载均衡的对外提供服务，当业务压力增加的时候，可通过向后端服务器池中添加主机来提升性能；当压力降低时，可减少主机，由于服务的 IP 不会发生变化，整个伸缩过程对访问者不产生影响。
  • 高安全需求服务
    业务统一通过负载均衡的对外提供服务，业务对应云主机不可直接从公网访问，充分保证安全性。
